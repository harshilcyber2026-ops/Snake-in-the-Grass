⚙ Reverse Engineering · Challenge #04

Snake in the Grass

A heavily obfuscated Python script checks a password and prints the flag. Unravel the layers.

Description

You've intercepted a Python script from a suspicious server. It prompts for a password, and only reveals the flag when the correct one is entered.

The script uses multiple layers of obfuscation to hide the password — lambdas, encoding chains, and hash verification. Your job is to reverse the logic without brute-forcing it.

No external dependencies. Just Python 3 and your brain. Read carefully — every variable name is a red herring.

Approach

Step 1 — Read the script top to bottom. Identify each obfuscation layer individually.

Step 2 — Trace the credential variable _k backwards through the transformation chain.

Step 3 — Once you have the password, run the script locally and enter it. The flag will be revealed.

Hints

_b64 is applied first, then _ugh. That means: _k = rot13( base64_decode("aGFlNGkzeV96Mw==") ). Decode the base64 string first, then apply ROT13 to the result.
The ___ list stores the flag XOR'd with key 0x5A. The function _xo(buf, key) decodes it. You don't need this to get the password — only the script needs it.
Open a Python shell and run exactly:
import base64, codecs
codecs.decode(base64.b64decode("aGFlNGkzeV96Mw==").decode(), 'rot_13')
That gives you the password directly.
